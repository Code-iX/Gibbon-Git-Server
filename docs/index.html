The installation process is straightforward, allowing you to manage everything through a clean and intuitive graphical interface. Utilize your existing Windows infrastructure as Gibbon Git Server runs seamlessly on IIS. Explore the demo and features to see it in action.

History

Gibbon Git Server is a fork of the Bonobo Git Server (old GitHub page).

After the Bonobo Git Server project was discontinued, Gibbon Git Server was upgraded to .NET 8 and continues to receive new features and improvements.
